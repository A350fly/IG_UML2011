Manuel d'utilisation d'umlTool 1.0

I) Création de classes :

  1. Cliquer sur le bouton 'Nouvelle Classe'.
  2. Cliquer sur la surface de dessin.
  3. Choisir un nom.
  4. Ajouter des attributs ou des méthodes.

II) Création d'interfaces :

  1. Cliquer sur le bouton 'Nouvelle interface'.
  2. Cliquer sur la surface de dessin.
  3. Choisir un nom.
  4. Ajouter des méthodes.

III) Ajout de méthodes :

  1. Spécifier toutes les informations nécessaires à votre méthode.
  2. Cliquer sur 'Ajouter méthode' pour créer une nouvelle méthode dans la liste courante.
  3. Une fois terminé, cliquer sur 'Valider'.

IV) Ajout attribut :

  1. Spécifier toutes les informations nécessaires de votre attribut.
  2. Cliquer sur 'Ajouter attribut' pour créer un nouvel attribut.
  3. Une fois terminé, cliquer sur 'Valider'.

V) Ajout de liens entre les classes ou les interfaces :

  1. Choisissez l'un des liens à établir.
  2. Le sens de création d'un lien dépend de l'ordre de sélection des objets : un objet sélectionné apparaît en rouge.
  3. Cliquer dans le vide pour déselectionné un objet.

VI) Générer du code java, c++ ou sql :

  1. Génération > Générer le code...
  2. Choisir le langage.
  3. Choisir le dossier de destination

VII) Exporter au format .png :

  1. Fichier > Exporter le graphe...
  2. Choisir le dossier de destination. Le fichier créé est diagramme.png
